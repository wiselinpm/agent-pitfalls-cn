# promptdiff — lint and auto-fix LLM prompts, ships as a Claude Code hook

Built a CLI tool that treats `.prompt` files as structured code and catches bugs before they ship.

**Claude Code integration:** Ships as a PostToolUse hook — when Claude writes a `.prompt` file, the hook auto-lints it. Errors block the edit with feedback, Claude self-corrects.
