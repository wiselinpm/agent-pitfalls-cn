## 摘要

<p>You can now store JSON values in <a href="https://vercel.com/docs/flags/vercel-flags">Vercel Flags</a>, extending the existing support for boolean, string, and number values. This allows you to collapse what used to take several related flags into a single feature flag.</p><p>
