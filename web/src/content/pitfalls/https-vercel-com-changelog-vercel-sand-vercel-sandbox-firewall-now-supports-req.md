## 摘要

<p>The <a href="https://vercel.com/docs/vercel-sandbox/concepts/firewall">Vercel Sandbox firewall</a> now supports forwarding specific HTTP requests to a proxy you control. You can also use matchers to filter forwarding and credentials brokering to only the requests that need it.
